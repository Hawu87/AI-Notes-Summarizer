# AI Note Summarizer

A production-ready full-stack web application that helps users turn long, unstructured notes into clear, concise summaries using AI.

## Overview

AI Note Summarizer is built to demonstrate real-world application development with modern web technologies. The project emphasizes secure authentication, private user data, server-side AI integration, and production-ready deployment practices.

Users can create personal notes, generate AI-powered summaries, and manage their content securely within an authenticated environment.

## Features

- Create and manage personal notes
- AI-powered summarization for long text
- Secure user authentication with email verification
- Private, user-scoped data using Row Level Security (RLS)
- Server-side OpenAI API integration (API keys never exposed)
- Dark mode support
- Responsive, modern UI

## Tech Stack

- **Frontend:** Next.js 16 (App Router), TypeScript
- **Styling:** Tailwind CSS, shadcn/ui
- **Backend:** Supabase (Database, Authentication, RLS)
- **AI:** OpenAI API
- **Deployment:** Vercel
